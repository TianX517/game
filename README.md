# 束对战游戏

一款同步选招的回合制对战游戏，基于"束"（粒子束）碰撞的机制：同属性互相抵消，不同属性可以继续前进直到命中。

## 在线试玩

部署在 Vercel（链接部署完成后填入）。

## 玩法

- **初始**：HP 3、能量 0。HP 归零即败（HP 无上限）。
- **每回合**：双方秘密选招，同步揭晓。
- **核心机制**：
  - 回能 / 物理防御 / 魔法防御：基础动作
  - 普通物理/魔法攻击：1 费 1 伤害
  - 同属性互相指 → 抵消（束对撞）
  - 不同属性 → 各自命中

## 角色

目前 6 个可选角色：
- **默认战士** — 全通用动作
- **三刀流侠客**（索隆）— 4 把刀 + 积气护盾被动
- **铁皮人** — 铁皮防御代替普通防御 + 崩/吸/投掷
- **疯子法师** — 连击倍率 + 血量触发免费券 + 负面状态吸血
- **刺客** — 飞镖/撒毒 + 遁入暗影 → 致命毒药/木偶操纵
- **阿宅** — 可乐盾 + 恰到好处压制

每个角色拥有不同的技能池与被动机制。详细规则见游戏内更新公告与技能悬浮提示（按住 Ctrl 查看机制详情）。

## 本地运行

双击 `index.html` 即可（浏览器直接打开）。如果遇到跨域问题（Chrome 有时会阻止 `file://` 下的本地 JS 加载），可用任意静态服务器：

```bash
# 任选其一
npx serve .
python -m http.server 8000
```

## 文件结构

```
index.html          - 引擎 + UI（主入口）
ai.js               - AI 策略
data/
  base.js           - 基础动作
  char_*.js         - 每个角色一个文件
  changelog.js      - 更新日志
```

加新角色只需新建一个 `data/char_xxx.js`，在 `index.html` 加一行 `<script src>`。
